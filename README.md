## 🛠️ Teknolojiler

- **Mikrodenetleyiciler**: STM32F103, STM32F4, Arduino
- **Sensörler**: BMI-088, ICM-42688-P, UKB sensörleri
- **İletişim Protokolleri**: I2C, SPI, UART
- **PCB Tasarım**: KiCad, Eagle
- **Programlama**: C, C++, Arduino
- **Güç Yönetimi**: LM2596, XL6009
